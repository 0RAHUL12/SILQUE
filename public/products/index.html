How To Choose

Match the format to the moment of use.

If guests hold drinks

Start with 8x8 cocktail napkins. They fit compact service moments without making the counter or tray feel crowded.

If guests sit for a meal

Start with 16x16 dinner napkins. They give the table more body, fold presence, and absorbency for longer service.

If cutlery is pre-set

Start with pocket cutlery napkins. They keep cutlery organised and make buffet or banquet setup faster.
